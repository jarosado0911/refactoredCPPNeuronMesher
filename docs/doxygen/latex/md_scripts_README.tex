A lightweight 3D Open\+GL-\/based viewer to render {\ttfamily .swc} neuron morphology files with interactive features\+:
\begin{DoxyItemize}
\item Rotation (mouse drag)
\item Zoom (scroll wheel)
\item Render mode selection (keys {\ttfamily 1} to {\ttfamily 6})
\item Colored nodes and edges by type
\item Variable thickness based on radius
\item Optional lighting and bounding box
\end{DoxyItemize}
