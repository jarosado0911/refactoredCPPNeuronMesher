This file provides Python bindings for the \mbox{\hyperlink{classNeuronGraph}{Neuron\+Graph}} class and \mbox{\hyperlink{structSWCNode}{SWCNode}} structure, enabling Python users to access the full functionality of the CPPNeuron\+Mesher library. The bindings expose all major operations including file I/O, morphology analysis, preprocessing, mesh operations, and trunk analysis.

This function defines the Python module \textquotesingle{}neurongraph\textquotesingle{} and exposes all C++ classes and methods to Python. The module provides two main classes\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{structSWCNode}{SWCNode}}\+: Represents individual neuron nodes with spatial and topological data
\item \mbox{\hyperlink{classNeuronGraph}{Neuron\+Graph}}\+: Main class for neuron morphology analysis and processing
\end{DoxyItemize}

All method signatures are preserved from C++ to Python, with automatic type conversion handled by pybind11 for standard types (int, double, string, maps, vectors). Python binding for \mbox{\hyperlink{structSWCNode}{SWCNode}} structure

Exposes the \mbox{\hyperlink{structSWCNode}{SWCNode}} C++ structure to Python, providing direct access to all neuron node properties. \mbox{\hyperlink{structSWCNode}{SWCNode}} represents a single point in a neuron morphology with spatial coordinates, topological relationships, and morphological type.
